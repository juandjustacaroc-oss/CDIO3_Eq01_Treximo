# 04_Dossier

## Documentación Final y Validaciones

Esta carpeta contiene la documentación formal del proyecto en su versión consolidada.

Incluye:

- Informe técnico final
- Arquitectura del sistema
- Matriz de Defensa de Diseño (MDD)
- Validaciones experimentales
- Resultados de pruebas
- Conclusiones
- Anexos fotográficos

---

## Contenido Esperado

- Manual de usuario
- Manual técnico
- Evidencias de auditoría
- Tablas de rendimiento
- Gráficas de frecuencia cardíaca
- Comparativas de medición

---

## Propósito

Respaldar académicamente el desarrollo del proyecto bajo metodología CDIO, demostrando:

- Justificación de decisiones técnicas
- Evaluación de riesgos
- Validación experimental
- Funcionalidad operativa del sistema

Esta sección representa la evidencia formal del proyecto ante evaluación.
